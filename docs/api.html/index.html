▸ overwriteConfigFile(config)

Overwrites the configuration file on disk if the "overwritable" option is set to true. NOOP when executing in the browser.

Parameters

Sets the global Logia configuration. It doesn't overwrite the configuration file on disk.

config(config: object)
Parameters
config (object) An object that defines any of the LogiaConfig properties
▸ slaveOf(url)

Enables fullstack mode. Logia configuration is dictated by the specified Logia server. NOOP when executing in the server.

slaveOf(url: string)
Parameters
url (string) Url that points to a running Logia server
▸ server.on(event, callback)

Listens for certain server events. Currently the only supported event is the "log" event which is triggered when Logia server receives a log.

server.on(event: string, callback: function)

callback (function)
